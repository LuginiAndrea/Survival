PhysicalEntity:::: (Definisce le entità basi fisiche, con le proprietà di distruzione, collisione, movimento e rotazione)
	-Il suo modello nella scena : Object3D
	-Nome : string
	-Distruttibile : bool  (Per entità vive se può essere killato)
	-HitsToDestruct : int (Vita per entità vive) (o per alcuni oggetti, quanto prima di essere distrutti)
	-CollisionOn : bool (=true)
	-CollisionOffGeometryOn : bool (=false)
	-CollisionOffGeometry : Object3D (Tipo porta) (= null se non c'è)
	-PlayerMove: bool (=false)
	

	-Costruttore(geometry, texture, color, name = "baseEntity", collisionOn = false, collisionOffAreaOn = false, collisionOffGeometry = null)
	-Distruttore () 
		:Liberiamo geometria e materiale (non la texture!)
	-Move(x,y,z)
	-Rotate(xAng,yAng,zAng)
	-OnDestruction() (=null per entità non distruttibili)
	-whileDestructing() (=null per entità non distruttibili) /
